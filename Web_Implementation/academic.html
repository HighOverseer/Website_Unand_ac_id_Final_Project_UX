beragam pilihan program studi universitas andalas

Universitas Andalas memiliki banyak pilihan program studi dan bidang keahlian yang tersebar di 15 fakultas dan program pascasarjana. Berbagai pilihan program studi terakreditasi nasional dan internasional memberikan pilihan bagi mahasiswa Universitas Andalas untuk dapat bersaing di tingkat nasional dan global.

program sarjana dan diploma

Universitas Andalas menawarkan banyak program studi pilihan untuk jenjang Sarjana dan Diploma yang tersebar di 15 fakultas

Program Magister dan Doktoral

Universitas Andalas menawarkan berbagai bidang keahlian untuk studi lanjut Magister dan Doktor.

Program Profesi

Untuk menunjang kompetensi dan daya saing lulusan, Universitas Andalas juga menyediakan pilihan Program Profesi, seperti Profesi Dokter, Apoteker, Nurse, Insinyur, dan lainnya.
